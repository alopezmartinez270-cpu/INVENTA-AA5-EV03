Registra primero un cliente y los productos disponibles. Después crea una venta seleccionando el cliente y agregando productos al detalle. INVENTA calcula el total y descuenta las existencias automáticamente.

Endpoints disponibles

Referencia rápida para realizar las pruebas en Postman.

MétodoEndpointDescripción
GET/api/healthVerifica el estado de la API.
POST/api/registroRegistra un usuario.
POST/api/loginValida credenciales.
GET / POST/api/clientesLista o crea clientes.
GET / PUT / DELETE/api/clientes/:idConsulta, actualiza o elimina un cliente.
GET / POST/api/productosLista o crea productos.
GET/api/productos/bajo-stock?limite=5Consulta productos que requieren reposición.
GET / PUT / DELETE/api/productos/:idConsulta, actualiza o elimina un producto.
GET / POST/api/ventasLista o registra ventas.
GET/api/ventas/:idConsulta una venta.
PUT/api/ventas/:id/anularAnula una venta y devuelve las existencias.

La documentación detallada se encuentra en docs/DOCUMENTACION_API.md y docs/openapi.yaml.
