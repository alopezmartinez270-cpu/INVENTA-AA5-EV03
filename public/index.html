INVENTA API

Servicios web para la gestion de clientes, productos y ventas.

Servicio activo

Endpoints disponibles

MetodoEndpointDescripcion
GET/api/healthVerifica el estado de la API.
POST/api/registroRegistra un usuario.
POST/api/loginValida credenciales.
GET / POST/api/clientesLista o crea clientes.
GET / PUT / DELETE/api/clientes/:idConsulta, actualiza o elimina un cliente.
GET / POST/api/productosLista o crea productos.
GET/api/productos/bajo-stock?limite=5Identifica productos que requieren reposicion.
GET / PUT / DELETE/api/productos/:idConsulta, actualiza o elimina un producto.
GET / POST/api/ventasLista o registra ventas.
GET/api/ventas/:idConsulta el detalle de una venta.
PUT/api/ventas/:id/anularAnula una venta y reintegra sus existencias.

La documentacion completa, cuerpos JSON, respuestas y codigos HTTP se encuentran en docs/DOCUMENTACION_API.md.
